# Businessweb

In this business web page, use a ready-made template for the frontend in boostrap, mount the backend server with Django 3.1, This business web application allows you to show the services offered, company history, contact record by mail, hours and a blog. The blog is registered and administered from the Django models through the ORM objects that connect to a sqlite3 database, this has two modelsâ€™ class; category models with 3 parameters, name, creation date and edition date. The other model, in this you are register the blogs, this has 8 parameters, title, content, publication date, image, author, category, creation date and edition date. Also, these use relationship Foreignkey and ManytoMany. The template tags are also used to apply inheritance to the html templates.
